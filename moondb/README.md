# Prodotto 
MOOn - Modulistica On Line
# Descrizione del prodotto 
Il progetto “MOOn” ha come obiettivo gestire tutto il ciclo della modulistica delle Pubblica Amministrazione dalla creazione del modulo alla relativa pubblicazione online.
**moondb**: ([Github repository](https://github.com/csipiemonte/moon/tree/main/moondb)): Script sql relativi al database utilizzato da Moon.

# Prerequisiti di sistema   
Database: Postgres 9.2  


# Installazione
- Eseguire lo script MOON-1.0.0_01_Creazione_Database.sql presente in **src/sql**

# Versioning
